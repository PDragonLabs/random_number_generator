# 🎲 VOID RNG Studio

**A beautiful, modern, and powerful Random Number Generator** with an awesome futuristic GUI.

Built as a single-file HTML tool — no installation needed. Just open and enjoy.

![VOID RNG Preview](https://via.placeholder.com/800x500/1a1a2e/6b21a8?text=VOID+RNG+Studio)

## ✨ Features

- **Stunning Glassmorphism UI** with neon accents
- **True Cryptographic Randomness** (using `crypto.getRandomValues`)
- **Advanced Controls** — Min/Max, decimals, multiple numbers
- **Dice Roller** — Roll up to 10 dice with visual results
- **Lottery Quick Picks** — Powerball, Mega Millions, EuroMillions styles
- **Live History** with timestamps
- **One-click Copy** + **Export to CSV**
- **Preset Ranges** (1-10, 1-100, 0-255, etc.)
- **Keyboard Support** (Spacebar / Enter to generate)
- **Confetti Explosions** on lucky rolls
- **Sound Effects** + Multiple Themes
- Fully responsive & mobile-friendly

## 📥 Installation

1. Download the file **`void-rng.html`**
2. Open it in your favorite browser
3. Done! 🎉

No build tools, no server, no dependencies.

## 📋 How to Use

- Enter **Min** and **Max** values
- Click **GENERATE** or press **Spacebar**
- Switch between **Standard**, **Dice**, and **Lottery** modes
- Click the **copy icon** to copy results
- Use **Export** to save your history as CSV
